🐦 Passive tweet watcher from the command line

Lurker is a simple terminal client for reading Y Combinator's Hacker News. It uses the public Hacker News API and is written entirely in Bash so it runs smoothly in the majority of terminal environments.

Lurker embraces the hacker in Hacker News by exposing a simple CLI for reading stories, comments, and user information. This makes it easy to read Hacker News while completing other command line activities, especially when combined with tools like tmux.

In addition to browsing Hacker News, lurker can summarize most text articles and snippets via another one of my tools, smoosh.
